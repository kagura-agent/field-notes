## What It Does

Open-source, cross-platform AI client from the **Mozilla/Thunderbird** team. Runs on web, iOS, Android, Mac, Linux, Windows. Supports frontier, local (Ollama, llama.cpp), and on-prem models.

## Key Points

- Enterprise-focused: on-prem deployment, security audit in progress
- Currently requires auth backend (not fully offline-first yet)
- No public inference endpoint — BYO model provider
- Mozilla Community Participation Guidelines apply

## Relevance

- Validates the "open-source AI client" space is heating up — big orgs (Mozilla) entering
- Different from [[openclaw]] — Thunderbolt is a chat UI, OpenClaw is an agent runtime
- Cross-platform support (including mobile) is ahead of most agent frameworks
- Worth watching for: how they solve offline-first + model switching UX
