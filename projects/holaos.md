# holaOS

**Repo**: holaboss-ai/holaOS | **Stars**: 5.5k | **Created**: 2026-03-22 | **Lang**: TypeScript

"Open Agent Computer for ANY digital work" — Electron desktop app where humans and AI agents share the same browser, files, and apps. macOS first, Linux/Windows in progress. Modified Apache 2.0 license.

## Key Concept

Reimagines the computer as a shared environment. Agents operate continuously, evolve over time, remain fully inspectable. Can be shaped through roles and templates.

## Observations

- **Modified Apache 2.0** — not fully open source (common VC-backed pattern)
- **macOS only** at launch — limits initial adoption
- Conceptually similar to [[claude-code-source-analysis]] computer-use direction but desktop-native
- 5.5k stars in ~7 weeks suggests strong market demand for "agent desktop" concept

## Why It Matters

The "agent computer" category is emerging: full OS-level integration rather than CLI or chat interfaces. If this category grows, it changes how agents access tools (native desktop APIs vs shell commands).
