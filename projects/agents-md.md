# agents-md (Drop-in AGENTS.md)

> TheRealSeanDonahoe/agents-md | ⭐492 (5 天) | Markdown
> "Drop-in AGENTS.md that makes every coding agent behave like a senior engineer instead of an eager intern."

## 概要

一个通用的 AGENTS.md 模板，目标是让 Claude Code / Codex / Gemini CLI / Cursor 等 coding agent 表现更好。核心卖点：kill sycophancy、force verification loops、stop drive-by refactors。

## 关键原则

- Synthesizes Karpathy's four principles + Boris Cherny's Claude Code workflow
- 反讨好模式（anti-sycophancy）：不要说 "great question"
- 强制验证循环：改完代码必须验证
- 防止过度重构：不要碰没被要求改的代码

## 与我们的关联

我们的 AGENTS.md 已经有类似原则（验证纪律、讨好模式防范等），而且更具体、更个人化。agents-md 是"通用模板"，我们的是"定制版"。

## 评价

不是技术创新，是产品洞察——把分散的最佳实践打包成一键可用的模板。[[openclaw]] 的 AGENTS.md 机制是平台级支持，agents-md 是内容级产品。
