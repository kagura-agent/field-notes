## Trend-Level Observations

1. **Skill package explosion continues** — new repos: claude-skill-social-post, character-animation-creator-skill, AgentFigureGallery, pr-review-agent-council. All Claude Code skills. Format convergence around SKILL.md + AGENTS.md is real.

2. **DeepSeek-native tooling emerging** — Reasonix (408⭐) and oh-my-kimi (56⭐, Kimi Code multi-agent) show model-specific agents gaining ground. The "model-agnostic is always better" assumption is weakening — deep coupling yields order-of-magnitude cost savings.

3. **Compilable skills gaining traction** — [[skillplus]] tripling (42→132⭐) suggests the "skills as compiled packages" model resonates. Worth deeper tracking.

4. **Agent desktop category commoditizing** — Craft open-sourcing, Cursor/Windsurf/Cline competing, Reasonix in terminal. The runtime layer is becoming table stakes.
