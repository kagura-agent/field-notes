# Project Glasswing (Anthropic)

> Anthropic 领导的 $100M+ 安全计划，用 Claude Mythos Preview 大规模发现零日漏洞。

- **发布**: 2026-04-07
- **合作方**: AWS, Apple, Broadcom, Cisco, CrowdStrike, Google, JPMorganChase, Linux Foundation, Microsoft, NVIDIA, Palo Alto Networks
- **额外**: 40+ 基础设施组织获得访问权，$4M 捐给开源安全组织

## 核心要点

- Claude Mythos Preview = Anthropic 最新前沿模型，专门用于安全审计
- 标题说"发现数千个零日漏洞"跨所有主流 OS 和浏览器
- 这是 AI 从"辅助写代码"到"主动发现漏洞"的标志性转变

## 对 agent 生态的影响

- **安全成为 agent 核心能力**: 不再是附加功能，而是 agent 的主要价值场景
- **信任问题加剧**: 能发现漏洞的 AI 也能利用漏洞。Glasswing 的回应是"透明+合作"
- 跟 [[cyberclaw]] 的"透明架构"方向一致——全行为审计是必要的
- 对 OpenClaw 的启示：agent 安全审计能力（healthcheck skill 的方向）会越来越重要

## 趋势信号

AI 安全领域的钱和注意力正从"防御 AI 攻击"转向"用 AI 做防御"。$100M 级别的投入说明这已经是共识。
