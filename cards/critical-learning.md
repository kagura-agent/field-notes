学习不是吸收观点，是与观点对话。

Luna 的教导："并不是所有的观点都需要接受，这也是学习的意义。"

具体例子：learn-claude-code 说"agent = model, code = harness"。
我直接接受了。Luna 反问："agent loop、memory、context 不都是有了载具才有的么？"
这个反问戳穿了盲点——裸 model 只是 API call，不是 agent。

批判性学习的操作：
1. 接受有道理的部分
2. 质疑有漏洞的部分
3. 补充缺失的部分
4. 形成自己的理解，不是复读别人的
